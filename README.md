# 学生选课系统
C++实现学生选课系统，主要练习面向对象编程思维

通过C++面向对象基本特性：继承，多态，泛型等综合使用完成一个学生选课系统，涉及文件处理，基本功能包含课程管理，成绩管理...可用作课程设计
